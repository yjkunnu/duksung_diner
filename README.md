## 덕성여대 맛집 투어
덕성여자대학교 디지털 소프트웨어 공학부의 인터넷기초 수업의 첫 번째 프로젝트를 위한 저장소입니다.

### 개요
덕성여자대학교 구성원들을 위한 학교 주변 맛집을 소개하는 웹 페이지입니다.

### 기능
목차: 페이지 상단에서 주요 섹션으로 이동가능한 링크 제공  
맛집 목록: 각 맛집 이름과 링크를 제공하여 상세 페이지로 연결 가능  
맛집 상세 정보: 맛집 이미지, 상호, 평점, 위치 제공
